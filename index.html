使用帮助

              
把 BKE 片段贴进左侧，右侧会显示精灵树和可回写的坐标属性。

            

            
          

          

            

              
1. 配置项目

              
点“项目”，选择有 config.bkpsr 的目录。图片会按 ImageAutoSearchPath 搜索，脚本里仍使用 BKE 风格相对路径。

            

            

              
2. 预览片段

              
支持 @sprite、@button、@buttonex、@layer、@textsprite、@addto、@anchor 和常用 @action。

            

            

              
3. 调坐标

              
点击画布或精灵树选中精灵，拖动会回写 pos=[x,y]；右侧属性面板可调整透明度、缩放、旋转、锚点和层级。

            

            

              
4. 发布包

              
开源 release 内置的示例资源由 Pillow 生成，不包含任何项目素材。Windows 可用 npm run pack:win 生成便携 EXE。
